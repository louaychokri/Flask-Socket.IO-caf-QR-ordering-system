# Flask-Socket.IO-caf-QR-ordering-system
QR Code based café ordering system built with Flask &amp; Socket.IO. Customers scan, view menus, order in real-time, while servers and admins manage orders through dedicated dashboards with live statistics.


# ☕ EL ANDALOUS — QR Café Ordering System

A modern QR-based ordering system for cafés and restaurants.  
Customers can scan a QR code, access the menu, and place orders in real-time.

---

## 🚀 Features

- 📱 QR Code access (no app needed)
- 🧾 Digital menu display
- ⚡ Real-time ordering (Socket.IO)
- 🌐 Web-based dashboard
- 🔄 Live updates between client & server
- 🧩 Simple and scalable architecture

---

## 🛠️ Tech Stack

- Python 3
- Flask
- Flask-SocketIO
- Flask-CORS
- SQLite (database)
